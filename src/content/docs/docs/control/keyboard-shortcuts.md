## Playback

| Shortcut | Action |
|---|---|
| `Space` | Play / pause |
| `Home` | Jump to start |
| `End` | Jump to end |

## Timeline

| Shortcut | Action |
|---|---|
| `Alt + scroll` | Zoom timeline (10 → 500 px/sec) |
| Drag ruler | Scrub playhead |

## Project

| Shortcut | Action |
|---|---|
| `Ctrl + S` | Save project |
| `Ctrl + Z` / `Ctrl + Y` | Undo / Redo |

## Windows

| Shortcut | Action |
|---|---|
| `Ctrl + L` | Toggle layout lock |

When layout is locked, windows can't be undocked or moved. Right-click a
window tab to dock / undock an individual window without unlocking
everything.
