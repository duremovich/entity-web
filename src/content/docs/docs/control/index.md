Two layers of control:

- **[OSC](/docs/control/osc/)** — inbound OSC, default namespace, port
- **[Bitfocus Companion](/docs/control/companion/)** — wire entity to a Stream Deck or X-keys
- **[Script command system](/docs/control/scripts/)** — JSON automation and test scripts
- **[Keyboard shortcuts](/docs/control/keyboard-shortcuts/)** — cheat sheet

OSC is the universal lingua franca. Companion is the universal adapter.
Scripts are for tests and repro.
