Four short steps from zero to a playing show:

1. **[Install](/docs/getting-started/install/)** — get entity onto your machine
2. **[Your first project](/docs/getting-started/first-project/)** — create a project folder and understand the layout
3. **[Importing media](/docs/getting-started/importing-media/)** — get clips into the media bin
4. **[Building a timeline](/docs/getting-started/building-a-timeline/)** — place clips, scrub, set sections, send output

Plan on about 20 minutes from a fresh install to a playing single-clip
project on your laptop display. Wiring projectors and calibration is a
separate session — [Projection](/docs/projection/).
