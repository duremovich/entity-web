Five concepts that, taken together, are entity:

- **[Projects on disk](/docs/concepts/projects-on-disk/)** — a project is a folder, not a file
- **[Timeline & clips](/docs/concepts/timeline/)** — tracks, clips, frame rates
- **[Sections (break-points)](/docs/concepts/sections/)** — named cue points on the timeline
- **[Screens vs physical outputs](/docs/concepts/screens-vs-outputs/)** — logical surfaces vs real displays
- **[Plugins](/docs/concepts/plugins/)** — two transport tiers, what they do

Read in order if you're new. Skim if you're not.
